Aria ist eine Client-App für Misskey, einer föderierten Social-Media-Plattform.

Unterstützt den Großteil von Misskeys Funktionen
Aria unterstützt die meisten Funktionen, die in Misskey verfügbar sind, einschließlich der Anzeige verschiedener Arten von Chroniken, Posten von Notizen, Bearbeiten von Clips, Listen oder Antennen, usw.

Zwischen mehreren Konten (mehrere Server) wechseln durch Wischen
Aria erleichtert es, verschiedene Chroniken anzuzeigen, indem man sie als Tabs speichert. Es können beliebige Chroniken von Benutzern als Tabs gespeichert werden, um verschiedenen Themen über Server hinweg zu folgen.

Design basiert auf dem offiziellen Misskey Web Client
Die Oberfläche ist so gestaltet, dass sie weitgehend der offiziellen Misskey-Oberfläche entspricht. Notizen können ebenfalls mit MFM gestaltet werden.

Umfangreich anpassbar
Zahlreiche Einstellungen, wie die Darstellungsgröße von Reaktionen und die Anzeige von Avataren, können geändert und den eigenen Vorlieben angepasst werden. Zudem kann das Anwendungsthema mit eigenen Farben angepasst werden.
